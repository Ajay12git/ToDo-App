A simple, full-stack ToDo application to manage tasks.  
Designed as a portfolio project to showcase **frontend, backend, and API integration** skills.

---

## Features

- Add, delete, and mark tasks as completed
- Persistent storage using `data.json` (backend)
- Interactive UI built with HTML, CSS, and JavaScript
- REST API built with Node.js and Express
- CORS enabled for frontend-backend communication

---

## Tech Stack

| Layer        | Technology           |
|-------------|--------------------|
| Frontend    | HTML, CSS, JavaScript |
| Backend     | Node.js, Express     |
| Database    | JSON file (`data.json`) |
| Utilities   | CORS, UUID           |
